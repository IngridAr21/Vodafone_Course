# WorldVert - Online Currency Converter

WorldVert is an online currency converter that I developed when I was 14 years old as part of Vodafone's free training program **#CodeLikeAGirl**. This was my very first project in the world of IT and marked my first steps into programming and web development.

## About #CodeLikeAGirl

#CodeLikeAGirl is a free training program by Vodafone designed to encourage girls to pursue careers in STEM and increase female representation in technology. The course covers web design and basic programming and is offered in multiple locations.

## About WorldVert

WorldVert is a simple and functional online currency converter that allows users to convert money between different currencies around the world. This project helped me learn and apply fundamental web technologies such as HTML, CSS, and JavaScript.

## Technologies used

- HTML  
- CSS  
- JavaScript
